# Changelog

All notable changes to the Tuist Grafana app plugin are documented here.

## 0.1.0 — Unreleased

- Initial release.
- Configuration page: Tuist server URL, account handle,
  `account:metrics:read` token, Prometheus datasource picker, scrape interval.
- Collector snippet generator for Grafana Alloy / Agent.
- Dashboards: Tuist Xcode and Tuist Gradle, each organised as
  Overview / Builds / Tests / Cache rows matching the Tuist project
  dashboard.
