Quarantined tests now have two modes: **Muted** (runs, failures masked — the existing behaviour) and **Skipped** (excluded from execution; the CLI passes the identifier to xcodebuild as `-skip-testing`). Pick the mode from the test case dropdown, filter by it on the Quarantined Tests page, or have an automation set it for you.
