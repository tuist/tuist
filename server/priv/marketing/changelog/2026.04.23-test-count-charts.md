The Test Cases and Flaky Tests dashboards now answer a question the numbers alone couldn't: *is this getting better or worse?* A new "Test cases" widget on the Test Cases page shows the number of distinct test cases with at least one run in the preceding 14 days at each point in the period, so you can watch your suite grow, and spot tests that quietly stopped running. On the Flaky Tests page, a matching "Flaky Tests" widget plots the number of currently-flagged flaky tests over time alongside the existing "Flaky Runs" metric — making it obvious when a push on quarantining (or a regression) moves the needle. Click either widget to swap the chart.
