---
title: Configurable default branch
category: "Product"
---

Tuist uses your project's default branch as the baseline for analytics, insights, and comparisons, from bundle size checks to test reliability and flaky test detection. Until now it was always assumed to be `main`. You can now set it from your project's settings, and when you connect a GitHub repository Tuist adopts the repository's default branch automatically.
