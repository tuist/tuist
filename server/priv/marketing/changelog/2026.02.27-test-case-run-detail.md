You can now click on any test case run to open a dedicated detail page with everything you need to understand what happened in that run. The overview tab shows run metadata (status, duration, branch, commit, scheme), and surfaces failures and crash reports inline. A separate Failures tab lets you drill into each failure with full error messages and repetition details when retry-on-failure is enabled. If the test was flaky, a Flaky Runs card groups all runs of that test for the same commit so you can see the full picture at a glance.
