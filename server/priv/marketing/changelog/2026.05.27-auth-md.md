---
title: Sign agents in with auth.md
description: "Connect agents to the Tuist MCP server without opening a browser, following the auth.md specification."
category: "Product"
---

Tuist now supports [auth.md](https://workos.com/auth-md), the open agent registration specification. Agents that implement the standard can connect to the Tuist MCP server directly: depending on the agent, Tuist either confirms the registration with the agent provider, or emails you a secure link with a six-digit code to read back to the agent. No browser-based OAuth round-trip required.
