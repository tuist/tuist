title: Exclude branches from test run filters
description: "Filter test runs by branches that do not contain a given substring."
category: "Product"

Test run filters now support excluding branches that contain a specific substring. Use the new "does not contain" branch operator in the dashboard, or the `git_branch_not_contains` API query parameter, to hide merge queue runs such as `gh-readonly-queue` while keeping the rest of the test history visible.
