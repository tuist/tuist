The Xcode Cache analytics page replaces the separate cache uploads and downloads widgets with a single Cache transfer widget, and adds two new widgets for cache latency and throughput. Each widget has a dropdown to switch the headline value between the combined view and either side of the split (downloads/uploads or read/write), and each chart renders the combined line on top of the per-action split with distinct colors per metric.
