The Usage section on your account dashboard surfaces the traffic Tuist Cache moves on your behalf. Three clickable widgets at the top — Egress, Ingress, and Requests — switch the chart between bytes downloaded from the cache, bytes uploaded to it, and the request count served, all in the selected window.

The time-series chart breaks each metric down per region so you can see where traffic concentrates, and the table below lists every cache node serving your account with its region, egress, ingress, and request totals. A Project dropdown scopes everything to a single project handle, and the date picker switches the resolution between hourly (for windows up to 36 hours) and daily.

The page only appears for accounts with Tuist Cache enabled; the underlying rollups are pushed from the managed cache mesh every minute and aggregated daily so historical queries stay fast.
