Self-hosted Tuist deployments can now integrate with a GitHub Enterprise Server instance. Switch to the **Enterprise server** tab in your organization's integrations, point Tuist at your GHES base URL, and install the app — comments, check runs, and repository connections all flow through your private instance from there on.

To enable it, register a GitHub App on your GHES instance and configure your Tuist server's `TUIST_GITHUB_APP_*` environment variables with its credentials. See the [self-host setup guide](/guides/server/self-host/install#platform-github) for the full list. Tuist Cloud (SaaS) does not support GHES yet — per-tenant GitHub App credentials are a separate workstream.
