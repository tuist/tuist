The Tuist GitHub app can now be installed on a self-hosted GitHub Enterprise Server instance. Switch to the **Enterprise server** tab in your organization's integrations, point Tuist at your GHES base URL, and install the app — comments, check runs, and repository connections all flow through your private instance from there on.

This requires your operations team to register a Tuist GitHub App on the GHES instance (with the same app name and credentials Tuist is configured with) before the install button will work end-to-end.
