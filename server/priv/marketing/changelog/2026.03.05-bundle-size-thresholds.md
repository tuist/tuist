Block pull requests when your app's bundle size grows beyond a configured limit. Tuist automatically creates GitHub Check Runs that compare the install or download size against a baseline branch. If the threshold is exceeded, you can accept the increase directly from the GitHub UI. Learn more in the [bundle insights documentation](https://tuist.dev/en/docs/guides/features/bundle-size#size-thresholds).
