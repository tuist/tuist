# Tuist Server Translation Context

The Tuist server is a web application providing a dashboard, marketing website, and documentation for Tuist -- a tool for managing and scaling Xcode projects and beyond.

## Translation Domains

- **marketing**: Public-facing marketing website. Tone should be engaging and persuasive.
- **dashboard**: Main application interface for logged-in users. Tone should be concise and clear.
- **dashboard_account**: Account settings, billing, and organization management.
- **dashboard_auth**: Authentication flows (login, signup, password reset).
- **dashboard_builds**: Build analytics and insights.
- **dashboard_cache**: Binary caching features.
- **dashboard_integrations**: Third-party integrations (GitHub, Slack, etc.).
- **dashboard_previews**: App preview deployment features.
- **dashboard_projects**: Project management features.
- **dashboard_tests**: Test analytics and reporting.
- **dashboard_gradle**: Gradle build integration.
- **dashboard_slack**: Slack integration.
- **docs**: API documentation pages.
- **errors**: Error messages shown across the application.
- **default**: General-purpose translations used across all areas.

## Plural Forms by Locale

When translating plural forms, use the correct number of plural forms for the target locale:
- Spanish (es): 2 plural forms
- Japanese (ja), Korean (ko), Chinese (zh_Hans, zh_Hant), Cantonese (yue_Hant): 1 plural form (no distinction)
- Russian (ru): 3 plural forms
