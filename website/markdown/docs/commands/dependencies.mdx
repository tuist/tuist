# Dependencies

Commands for pulling and updating third-party dependencies defined in `Dependencies.swift` manifest file.

The dependencies will be saved in `/Tuist/Dependencies` directory.

You can read more [here](/docs/dependencies/dependencies/).

## Commands

### Fetching

```bash
tuist dependencies fetch
```

<ArgumentsTable
  args={[
    {

  ]}
/>

### Updating

```bash
tuist dependencies update
```
