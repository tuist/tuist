# Generate documentation

Leveraging on [swift-doc](https://github.com/SwiftDocOrg/swift-doc), we facilitate a convenient way to generate documentation for your Xcode projects. It is as simple as running:

```bash
tuist doc --path /path/to/Project.swift MyApp
```

## Modes

### localhost
Tuist generates an html website that you can navigate with your browser.

```bash
tuist doc --path /path/to/Project.swift MyApp --localhost
```

### files-only
Tuist generates markdown documentation in a temporary directory.


```bash
tuist doc --path /path/to/Project.swift MyApp --files-only
```
