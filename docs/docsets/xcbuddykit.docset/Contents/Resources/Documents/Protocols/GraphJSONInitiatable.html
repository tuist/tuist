GraphJSONInitiatable

protocol GraphJSONInitiatable: AnyObject

The objects that conform this protocol are nodes that can be initialized with its JSON representation, the project path, and the graph loader context.

  • init(json:projectPath:context:)

    Default constructor.

    Throws

    an error when the object cannot be initialized

    Declaration

    Swift

    init(json: JSON,

    Parameters

    json

    json representation of the object that is going to be initialized.
    projectPath

    path to the folder where the project’s definition that contains this object is.
    context

    graph loader context.
