# Kod {#code}

Tuist jest monorepo. Każdy komponent ma własne narzędzia, zależności i lokalne
kroki uruchamiania. Wybierz obszar, w którym chcesz wnieść swój wkład:

- <LocalizedLink href="/contributors/code/cli">CLI</LocalizedLink>
- <LocalizedLink href="/contributors/code/server">Serwer</LocalizedLink>
- <LocalizedLink href="/contributors/code/handbook">Podręcznik</LocalizedLink>
- <LocalizedLink href="/contributors/code/docs">Dokumenty</LocalizedLink>

Wszystkie komponenty używają
<LocalizedLink href="https://mise.jdx.dev/getting-started.html">Mise</LocalizedLink>
do zarządzania zależnościami.
