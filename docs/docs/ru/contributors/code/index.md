# Код {#code}

Tuist — это монорепозиторий. Каждый компонент имеет свои собственные
инструменты, зависимости и локальные этапы выполнения. Выберите область, в
которую вы хотите внести свой вклад:

- <LocalizedLink href="/contributors/code/cli">CLI</LocalizedLink>
- <LocalizedLink href="/contributors/code/server">Сервер</LocalizedLink>
- <LocalizedLink href="/contributors/code/handbook">Справочник</LocalizedLink>
- <LocalizedLink href="/contributors/code/docs">Документация</LocalizedLink>

Все компоненты используют
<LocalizedLink href="https://mise.jdx.dev/getting-started.html">Mise</LocalizedLink>
для управления зависимостями.
