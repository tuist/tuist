# تسجيل الدخول {#logging}

تتبنى CLI واجهة [swift-log](https://github.com/apple/swift-log) للتسجيل. تقوم
الحزمة بتجريد تفاصيل تنفيذ التسجيل، مما يسمح لـ CLI بأن تكون محايدة تجاه الخلفية
الخلفية للتسجيل. يتم حقن المسجل بالاعتماد على المهام المحلية ويمكن الوصول إليه
من أي مكان باستخدام:

```bash
Logger.current
```

:::: المعلومات
<!-- -->
لا تنشر المهام المحلية القيمة عند استخدام `Dispatch` أو المهام المنفصلة، لذا إذا
كنت تستخدمها، فستحتاج إلى الحصول عليها وتمريرها إلى العملية غير المتزامنة.
<!-- -->
:::

## ما يجب تسجيله {#what-to-log}

السجلات ليست واجهة مستخدم CLI. إنها أداة لتشخيص المشكلات عند ظهورها. لذلك، كلما
زادت المعلومات التي تقدمها، كان ذلك أفضل. عند إنشاء ميزات جديدة، ضع نفسك في مكان
المطور الذي يواجه سلوكًا غير متوقع، وفكر في المعلومات التي قد تكون مفيدة له.
تأكد من استخدام [مستوى السجل]
الصحيح(https://www.swift.org/documentation/server/guides/libraries/log-levels.html).
وإلا لن يتمكن المطورون من تصفية الضوضاء.
