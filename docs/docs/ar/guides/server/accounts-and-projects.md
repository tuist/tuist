# الحسابات والمشاريع {#accounts-and-projects}

تتطلب بعض ميزات Tuist خادمًا يضيف استمرارية البيانات ويمكنه التفاعل مع الخدمات
الأخرى. للتفاعل مع الخادم، تحتاج إلى حساب ومشروع تربطه بمشروعك المحلي.

## الحسابات {#accounts}

لاستخدام الخادم، ستحتاج إلى حساب. هناك نوعان من الحسابات:

- **الحساب الشخصي:** يتم إنشاء هذه الحسابات تلقائيًا عند التسجيل ويتم تحديدها
  بواسطة اسم مستخدم يتم الحصول عليه إما من مزود الهوية (مثل GitHub) أو من الجزء
  الأول من عنوان البريد الإلكتروني.
- **حساب المؤسسة:** يتم إنشاء هذه الحسابات يدويًا ويتم تحديدها بواسطة اسم مستخدم
  يحدده المطور. تسمح المؤسسات بدعوة أعضاء آخرين للتعاون في المشاريع.

إذا كنت على دراية بـ [GitHub](https://github.com)، فإن المفهوم مشابه لمفهومهم،
حيث يمكنك إنشاء حسابات شخصية وحسابات مؤسسية، ويتم تحديدها بواسطة معرف ** الذي
يتم استخدامه عند إنشاء عناوين URL.

::: info CLI-FIRST
<!-- -->
تتم معظم عمليات إدارة الحسابات والمشاريع من خلال واجهة CLI. نحن نعمل على تطوير
واجهة ويب ستسهل إدارة الحسابات والمشاريع.
<!-- -->
:::

يمكنك إدارة المؤسسة من خلال الأوامر الفرعية الموجودة ضمن
<LocalizedLink href="/cli/organization">`tuist organization`</LocalizedLink>.
لإنشاء حساب مؤسسة جديد، قم بتشغيل:
```bash
tuist organization create {account-handle}
```

## المشاريع {#projects}

يجب أن تكون مشاريعك، سواء كانت مشاريع Tuist أو مشاريع Xcode الأولية، مدمجة مع
حسابك من خلال مشروع بعيد. وبمواصلة المقارنة مع GitHub، فإن الأمر يشبه وجود
مستودع محلي ومستودع بعيد حيث تقوم بدفع التغييرات الخاصة بك. يمكنك استخدام
<LocalizedLink href="/cli/project">`tuist project`</LocalizedLink> لإنشاء
المشاريع وإدارتها.

يتم تحديد المشاريع بواسطة معرف كامل، وهو نتيجة لربط معرف المؤسسة ومعرف المشروع.
على سبيل المثال، إذا كان لديك مؤسسة بمعرف `tuist` ، ومشروع بمعرف `tuist` ،
فسيكون المعرف الكامل هو `tuist/tuist`.

يتم الربط بين المشروع المحلي والمشروع البعيد من خلال ملف التكوين. إذا لم يكن
لديك ملف، فقم بإنشائه في `Tuist.swift` وأضف المحتوى التالي:

```swift
let tuist = Tuist(fullHandle: "{account-handle}/{project-handle}") // e.g. tuist/tuist
```

::: warning TUIST PROJECT-ONLY FEATURES
<!-- -->
لاحظ أن هناك بعض الميزات مثل
<LocalizedLink href="/guides/features/cache">التخزين المؤقت
الثنائي</LocalizedLink> التي تتطلب أن يكون لديك مشروع Tuist. إذا كنت تستخدم
مشاريع Xcode الأولية، فلن تتمكن من استخدام هذه الميزات.
<!-- -->
:::

يتم إنشاء عنوان URL لمشروعك باستخدام المعرف الكامل. على سبيل المثال، يمكن الوصول
إلى لوحة معلومات Tuist العامة على
[tuist.dev/tuist/tuist](https://tuist.dev/tuist/tuist)، حيث `tuist/tuist` هو
المعرف الكامل للمشروع.
