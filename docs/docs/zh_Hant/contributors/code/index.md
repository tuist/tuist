# 程式碼{#code}

Tuist 採用單一儲存庫架構。每個元件皆具備專屬工具集、依賴項及本地執行步驟。請選擇您欲貢獻的領域：

- <LocalizedLink href="/contributors/code/cli">CLI</LocalizedLink>
- <LocalizedLink href="/contributors/code/server">伺服器</LocalizedLink>
- <LocalizedLink href="/contributors/code/handbook">手冊</LocalizedLink>
- <LocalizedLink href="/contributors/code/docs">文件</LocalizedLink>

所有元件皆採用
<LocalizedLink href="https://mise.jdx.dev/getting-started.html">Mise</LocalizedLink>
進行依賴管理。
